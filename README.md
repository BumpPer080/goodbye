# Goodbye Bostech

A heartfelt farewell web application designed for the Bostech team. This interactive experience allows team members to unlock personalized messages and memories.

![Chair](./public/imgs/chair.jpg)

## Features

- **Interactive Profile Selection**: Choose your profile to begin.
- **PIN Protection**: Secure access to personal messages.
- **Dynamic Animations**: Profiles feature unique icon animations (wiggles, pulses, bounces).
- **Special Cards**: Personalized "Thank You" cards for each team member.
- **Interactive Elements**: Like buttons with floating heart animations.
- **Atmospheric Backgrounds**: Cinematic and confetti effects for a memorable experience.

## Getting Started

1.  Clone the repository.
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Run the development server:
    ```bash
    npm run dev
    ```

## Technologies Used

- React
- TypeScript
- Tailwind CSS
- Framer Motion (or CSS animations)
- Vite
